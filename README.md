# UserControl-NodeJS

Backend do sistema de controle de veiculos na instituição de ensino.

- Nível de acesso administrativo (999) - Tem acesso ao Dashboard Administrativo.
- Nível de acesso Aluno (1) - Tem acesso ao Dashboard do aluno.
- Nível de acesso Gestor (2) - Tem acesso ao Dashboard do gestor.

## Importante

- Necessário todas as dependências do projeto
- Necessário informações para conexão da base de dados

### Requisitos

- Node
- yarn

### Iniciando

```
$ git clone https://github.com/biancabarbosa23/vehicleControl-NodeJs.git
```

```
$ cd vehicleControl-NodeJs
```

```
$ yarn install
```

```
$ yarn dev
```
